StringBuilder

定义

在 C# 中，string 是不可变的(immutable)，每次修改都会创建新的字符串对象。。而StringBuilder则是可变的。

命名空间

StringBuilder 位于 System.Text 命名空间，使用时需要添加 

using System.Text;

常规用法


StringBuilder sb = new StringBuilder();
// 可以添加各种类型
sb.Append("Hello");
sb.Append(123);
// 指定位置插入
sb.Insert(7, "World");
// 替换字符串
sb.Replace("World", "C#");
// 删除
sb.Remove(0, 5);
// 清空
sb.Clear();
// 转换类型
string result = sb.ToString();
